CbE_build-template
==================

Modelo de build para projetos Java.
Desenvolvido por Guilherme Lacerda e Carlos Lopes (www.codingbyexample.org)

Neste template, esta configurado as seguintes definicoes/ferramentas:
- Exclusao de pastas/arquivos temporarios
- Criacao de pastas de documentacao
- Compilacao de fontes
- Compilacao de testes
- Geracao de relatorios de teste do JUnit
- Code Coverage com Emma e JaCoCo
- Analise de codigo com PMD
- Verificacao de padroes com Checkstyle
